💡 Como Usar

<h1 class="text-3xl font-bold text-primary mb-4">Título</h1>

Classes principais disponíveis:

  • text-{size} - Tamanhos de fonte
  • font-{weight} - Pesos de fonte
  • text-{color} - Cores de texto
  • btn-modern - Botões modernos
  • card-modern - Cards modernos
